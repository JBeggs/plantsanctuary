# Plant Sanctuary

- Python3.8 or Greater
- Django==3.2.19
- django-allauth latest
- django-cms==3.9.0

* djangocms-file==3.0.0
* djangocms-googlemap==2.0.0
* djangocms-icon==2.0.0
* djangocms-installer==2.0.0
* djangocms-link==3.1.1
